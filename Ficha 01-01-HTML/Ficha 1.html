HTML- HyperText Markup Language

Publicado por Marco Olim, 10 de octubro2021

Os documentos em HTML são arquivos de texto simples que podem ser criados e editados em qualquer editor de textos comum.

Juntamente com o CSS e o Javascript, o HTML é a base de todos os websites e aplicações web.

O que e o HTML?

O que é o HTML? HTML (abreviação para a expressão inglesa HyperText Markup Language, que significa: "Linguagem de Marcação de Hipertexto" é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores.
